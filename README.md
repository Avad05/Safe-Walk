# 🚶‍♀️ Safe Walk – Community Route Sharing App

Safe Walk is a community-driven web application that helps people share walking routes they feel safe using—especially at night. Unlike traditional navigation apps, Safe Walk focuses on **perceived safety based on real user experiences**.

---

## ✨ Features

### 🗺 Route Sharing
- Record and share walking routes
- Rate lighting, foot traffic, and visibility
- Add safety notes and time-of-day context
- Tag routes (well-lit, solo-woman safe, accessible)

### 👥 Community Verification
- Routes gain trust as more users walk them
- Flag outdated or unsafe routes

### 🚨 Companion Safety (Planned)
- Share live route with trusted contacts
- Auto-alerts if movement stops or deviates
